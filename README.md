# LexConnect

AI-powered legal assistance platform that helps clients find lawyers and get insights from Indian civil case judgments.

## Features

- AI Legal Assistant (RAG using civil case judgments)
- Lawyer recommendation system
- Client case management
- Lawyer dashboard
- React frontend + FastAPI backend
